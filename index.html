About TestCase Genius

TestCase Genius is an AI-powered manual test case generator for QA teams. Paste any feature description and get a complete suite of test cases — positive, negative, boundary, edge, security, accessibility, and more — each with a rationale explaining why it matters.

Features

  • 3 AI engines: offline Demo, free Google Gemini, paid OpenAI
  • Adjustable depth (8 to 30+ cases) and audience tone
  • Filter by type or priority
  • Export to Excel, CSV, Markdown, JSON, and Jira CSV
  • One-click push to Jira (via local proxy or Cursor MCP)

Tech

Vanilla HTML/JS · Tailwind CSS · SheetJS for Excel · Node.js proxy for Jira push · Cursor Skills for MCP integration
